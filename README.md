<b>Django E-commerce Platform 🛒</b>

A full-featured E-commerce web application built with Python and Django. Users can browse products, manage shopping carts, and complete a simulated checkout. This project demonstrates backend development, REST API integration, database management, responsive frontend design, and deployment readiness.

✨ Features

User authentication: registration, login, logout

Product catalog with categories and search functionality

Shopping cart & order management

RESTful API integration for frontend-backend communication

Database operations using Django ORM (SQLite/PostgreSQL)

Responsive design using HTML, CSS, and Bootstrap

Deployment-ready for Heroku/PythonAnywhere

🛠️ Technologies Used

Python 3.x | Django 4.x | HTML | CSS | Bootstrap | SQLite/PostgreSQL | Git/GitHub


🎯 Skills Demonstrated

Python & Django development

REST API creation & integration

Database management with Django ORM

Responsive frontend design

Deployment & version control
